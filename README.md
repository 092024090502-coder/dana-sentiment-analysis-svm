# DANA App Review Sentiment Analysis using SVM

## 📌 Overview
This project performs sentiment analysis on user reviews of the DANA application collected from the Google Play Store. The model uses Support Vector Machine (SVM) with TF-IDF feature extraction to classify reviews into positive, negative, and neutral sentiments.

## 🎯 Objectives
- Collect user reviews from Google Play Store.
- Preprocess Indonesian text data.
- Build a sentiment classification model using SVM.
- Evaluate model performance.

## 🛠️ Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Sastrawi
- Google Play Scraper
- Matplotlib
- WordCloud

## 📂 Repository Contents
- `02_Scraping_Data_Sentimen_Analisis_Project.ipynb` - Data scraping
- `03_Pemodelan_Sentimen_Analisis_Project.ipynb` - Model training
- `04_Testing_Sentimen_Analisis_Project.ipynb` - Model testing
- `05_hasil_scraping_dana.csv` - Dataset
- `06_requirements.txt` - Python dependencies
- `07_svm_model.pkl` - Trained model
- `08_tfidf_vectorizer.pkl` - TF-IDF vectorizer

## 📊 Results
- Positive: **67.6%**
- Negative: **27.8%**
- Neutral: **4.6%**
